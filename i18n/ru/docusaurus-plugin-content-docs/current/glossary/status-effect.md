# <img src="/icons/status-effect.svg" alt="Status Effect Icon" /> Состояние

См. также: [Состояния](/docs/battles/status-effects/index).

Этот кубик навыка размещается особым образом — например, на враге или в клетке,
— а его способность применяется согласно соответствующей статье. Когда кубик
навыка с эффектом состояния убирается из своего текущего положения, этот кубик
истощается, если его способность не указывает иное.

Обратите внимание: [сюжетные боевые единицы](/docs/glossary/quest-unit) не могут
получать состояния от способностей [персонажей](/docs/glossary/adventurer).

**Снимаются ли состояния с персонажей автоматически после боя?**

Состояния не снимаются с [персонажей](/docs/glossary/adventurer) автоматически
после боя. Они остаются на [треке перезарядки](/docs/glossary/cooldown-track),
пока не будут сняты.

В [конце боя](/docs/battles/end-of-battle) состояния снимаются со всех остальных
[боевых единиц](/docs/glossary/unit), включая [врагов](/docs/glossary/enemy) и
[компаньонов](/docs/glossary/companion).

**В какой момент убираются кубики состояний?**

Общее правило: состояния снимаются после применения любых навыков врага,
срабатывающих **после поражения** этой боевой единицы.

<a href="https://discord.com/channels/273472391403798528/1361396124782694450/1387181293388693556" target="_blank">Источник</a>

---

**Может ли одно и то же состояние действовать несколько раз?**

Нет, у вас может быть только один экземпляр каждого состояния. Если
[персонаж](/docs/glossary/adventurer) должен получить кубик состояния с
символом, который уже есть на его треке перезарядки, кубик на треке перезарядки
убирается из текущей позиции и помещается в крайнее слева незанятое место.
